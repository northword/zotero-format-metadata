# 标题化学公式处理

## 描述

Automatically corrects chemical formulae in titles. Supports Charge Number (e.g. `Co2+` -> `Co<sup>2+</sup>`) and Stoichiometric Number (e.g. `Cu2O` → `Cu<sub>2</sub>O`)

## 示例

```diff json
{
  "title": "Co2+ in catalysis"
}
{
  "title": "Co<sup>2+</sup> in catalysis"
}
```
