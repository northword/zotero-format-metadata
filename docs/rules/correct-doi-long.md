# 长 DOI 转换

## 描述

If DOI is short form (e.g. `10/p6kd`), converts it to long form (e.g. `10.1016/j.fuel.2022.126104`).

## 示例

```diff json
{
  "DOI": "10/p6kd"
}
{
  "DOI": "10.1016/j.fuel.2022.126104"
}
```
