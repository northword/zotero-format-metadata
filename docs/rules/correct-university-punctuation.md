# 大学名标点规范化

## 描述

Normalizes university names.

For example, in Chinese entries, replaces half-width parentheses `()` with full-width ones `（）`.

## 示例

```diff json
{
  "university": "Peking University (Beijing)"
}
{
  "university": "Peking University（Beijing）"
}
```
