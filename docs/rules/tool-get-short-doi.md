# 短 DOI 获取工具

## 描述

Retrieves short DOI from the DOI, and stores it in the `extra.short-doi` field.

## 示例

```diff json
{
  "DOI": "10.1016/j.fuel.2022.126104",
  "extra": ""
}
{
  "DOI": "10.1016/j.fuel.2022.126104",
  "extra": "short-doi: 10/p6kd"
}
```
