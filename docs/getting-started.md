# 快速开始

本页面将指导你快速安装和使用 Zotero 格式化和清理元数据插件。

## 系统要求

- **Zotero 8.0** 或更高版本
- 任何支持 Firefox 扩展的操作系统（Windows、macOS、Linux）

## 安装

### 方式一：从 GitHub Release 安装（推荐）

1. 访问 [GitHub Release 页面](https://github.com/northword/zotero-format-metadata/releases)
2. 下载最新版本的 `.xpi` 文件
3. 在 Zotero 中打开 **首选项** > **高级** > **文件**
4. 点击 **"安装附加组件"** 按钮
5. 选择下载的 `.xpi` 文件
6. 重启 Zotero

### 方式二：从文件安装

1. 克隆或下载本项目源码
2. 运行 `npm run build` 构建（需要 Node.js）
3. 生成的 `.xpi` 文件位于 `build/` 目录
4. 按照方式一的步骤 3-6 安装

## 基本使用

### 首次启动

安装完成后，打开 Zotero 且打开任意条目时，会触发插件的初始化。你可以在 Zotero 的 **首选项** > **Linter** 中配置插件。

### 三种运行方式

插件提供多种灵活的运行方式：

#### 1. 自动扫描（标准 Lint 流）

当选中一个或多个条目时，插件会自动在后台执行这些规则：

- 适用于大多数规则
- 无需手动触发
- 在首选项中可配置是否启用自动运行

配置位置：**首选项** > **Linter** > **运行模式** > **自动扫描**

#### 2. 菜单工具

部分高级功能（如批量操作、交互式设置）只能通过菜单触发。

**访问方式**：在条目列表中右键菜单，或顶部菜单栏的工具菜单

常见工具包括：

- 🧹 **清理额外字段** — 移除非标准字段
- 🔄 **更新元数据** — 从网络源更新信息
- 🏷️ **设置语言** — 批量设置语言字段

#### 3. 手动执行

在 Zotero 首选项中，你可以为任何规则自定义快捷键或从菜单手动触发。

### 配置首选项

打开 Zotero，进入 **首选项** > **Linter** 标签页：

#### 规则启用/禁用

- 可选择启用或禁用特定规则
- 某些规则可能有额外的参数配置（如仅检测简体中文）
- 规则的冷却期设置，避免过于频繁的执行

#### 日志级别

设置插件的日志输出详细度：

- **Error** — 仅输出错误
- **Warning** — 输出错误和警告
- **Info** — 输出所有信息（默认）
- **Debug** — 最详细的调试信息

### 查看规则执行结果

在 Zotero 的 **工具** > **浏览器控制台** 中可以查看插件的实时日志输出。

## 常见场景

### 场景 1：修复新导入的批量条目

```
1. 从网络导入 100 条参考文献到 Zotero
2. 在条目列表中全选这些条目（Ctrl+A）
3. 等待插件自动运行规则进行检查和修复
4. 查看执行结果，确认无误
```

### 场景 2：清理已有库中的数据

```
1. 选择需要清理的条目或全库（Ctrl+A）
2. 右键菜单 > Linter > "清理额外字段"
3. 根据弹出的对话框选择要清理的字段类型
4. 确认后自动执行清理
```

### 场景 3：为所有论文添加简体中文标签

```
1. 选择需要处理的论文条目
2. 右键菜单 > Linter > "设置语言"
3. 选择"简体中文"并确认
4. 插件自动向这些条目的语言字段添加该语言
```

## 首选项配置详解

### 规则分类

插件的 40+ 规则分为以下几大类，可在首选项中分类查看：

| 分类           | 示例规则                   | 说明                   |
| -------------- | -------------------------- | ---------------------- |
| **条目级规则** | 无重复条目、自动类型识别   | 整体检查条目的有效性   |
| **标题规则**   | 句子式大小写、特殊格式处理 | 规范标题格式和内容     |
| **作者规则**   | 作者大小写、拼音分割       | 修正作者名格式         |
| **期刊规则**   | 期刊缩写、正式名称统一     | 标准化期刊信息         |
| **语言规则**   | 自动检测和设置语言         | 准确识别内容语言       |
| **其他字段**   | DOI 格式、页码范围等       | 修复其他字段的常见问题 |
| **工具类**     | 批量清理、元数据更新       | 提供强大的批量处理能力 |

## 故障排除

### 插件不工作

1. 确保已正确安装（在 **首选项** > **扩展** 中可见）
2. 检查插件是否启用（首选项中有启用/禁用选项）
3. 尝试重启 Zotero
4. 查看浏览器控制台（**工具** > **浏览器控制台**）中是否有错误信息

### 规则执行缓慢

1. 检查是否有过多规则同时运行
2. 在首选项中增加规则的冷却期
3. 尝试减少条目数量，分批处理
4. 检查是否有网络请求类的规则（如获取 DOI 信息）

### 修复结果不符合预期

1. 仔细阅读该规则的详细说明（见 [功能详情](/rules/)）
2. 检查该规则的配置选项
3. 向 [GitHub Issues](https://github.com/northword/zotero-format-metadata/issues) 报告问题
